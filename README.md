BTTool
===================


用于修改BT（种子）文件中的**敏感**信息。

----------


开发环境
-------------

这是一个用c# Winform程序，使用VS2013开发，请使用**VS2012或以上**的版本的Visual Studio打开。
同时需要安装有**.Net Framework3.5或以上**版本。

----------
